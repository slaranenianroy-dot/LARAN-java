#  Pizza Palace - Java Console Food Ordering System

A simple **Java console-based food ordering application** that allows users to browse food categories, place orders, select item sizes, calculate the bill, and apply discount coupons.

##  Features

-  Veg Pizza Menu
-  Non-Veg Pizza Menu
-  Burger Menu
-  Pasta Menu
-  Beverage Menu
-  Size Selection (Small, Medium, Large)
-  Quantity Selection
-  Automatic Bill Calculation
-  Coupon Discount System
-  Continue Ordering Option

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner Class for User Input

##  How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/Pizza-Palace.git
```

### 2. Navigate to the Project Folder

```bash
cd Pizza-Palace
```

### 3. Compile the Program

```bash
javac task.java
```

### 4. Run the Program

```bash
java project.task
```

##  Menu Options

```
1. Veg Pizza
2. Non Veg Pizza
3. Burgers
4. Pasta
5. Beverage
6. View Bill
7. Exit
```

Each category contains multiple food items.

After selecting an item, the user can:

- Choose the size
- Enter the quantity
- Continue shopping or view the bill
