#  Pizza Palace - Java Console Food Ordering System

A simple **Java console-based food ordering application** that allows
users to browse food categories, place orders, select item sizes,
calculate the bill, and apply discount coupons.

##  Features

-    Veg Pizza Menu
-    Non-Veg Pizza Menu
-    Burger Menu
-    Pasta Menu
-    Beverage Menu
-    Size Selection (Small, Medium, Large)
-    Quantity Selection
-    Automatic Bill Calculation
-    Coupon Discount System
-    Continue Ordering Option

##  Technologies Used

-   Java
-   Object-Oriented Programming (OOP)
-   Scanner Class

##  Project Structure

``` text
project/
├── task.java
└── README.md
```

##  How to Run

1.  Compile:

``` bash
javac task.java
```

2.  Run:

``` bash
java project.task
```

##  Menu Categories

-   Veg Pizza
-   Non Veg Pizza
-   Burgers
-   Pasta
-   Beverage

Users can choose item size, quantity, and continue ordering before
generating the final bill.
